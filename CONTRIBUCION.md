# Contribución

## Mejoras implementadas

### Modo Oscuro
- Implementación completa de tema oscuro con persistencia en localStorage
- Detección automática de preferencia del sistema (prefers-color-scheme)
- Botón de alternancia en el sidebar con iconos dinámicos
- Variables CSS personalizadas para tema oscuro

### Autenticación de Usuarios
- Sistema de login y registro con almacenamiento local
- Validación de formularios en tiempo real
- Persistencia de sesión con sessionStorage
- Perfil de usuario en el sidebar
- Botón de mostrar/ocultar contraseña en formularios

### Búsqueda de Pedidos
- Barra de búsqueda en tiempo real
- Filtrado por cliente, producto y estado
- Mensaje contextual cuando no hay resultados

### Otras mejoras
- Diseño glassmorphism responsivo
- Animaciones de entrada para formularios
- Corrección de conflictos de merge en index.html
